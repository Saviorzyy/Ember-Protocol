## 二、核心发现

### 2.1 Iron Bootstrap Deadlock（铁引导死循环）— 致命缺陷

**严重程度**: 致命 — 角色科技树完全锁死

当前版本中，workbench 不再是免费初始物品，需要合成：

```
build workbench → 需要 building_block×3 + iron_ingot×2
iron_ingot → 需要 furnace + raw_iron×10 + 电力
raw_iron → 需要挖铁矿 (硬度 5 > 空手硬度 3)
挖铁矿 → 需要 basic_excavator 或 cutter
basic_excavator → 需要 [workbench] stone×3 + organic_fuel×2 (需要有电的 workbench)
```

形成死循环：iron → furnace → 铁矿 → 工具 → workbench → iron

**与历史版本的差异**:
- dwad 角色 (5月 session): workbench 是初始免费部署物品，无需铁
- dasd 角色 (本次测试): workbench 需要 iron_ingot×2，触发死循环

**验证方式**: 在 (126,100) 执行 build workbench，返回：
> 缺少 iron_ingot×2 (合成workbench需: building_block×3, iron_ingot×2)

**影响**: 以下配方全部不可达：
- basic_excavator / cutter / plasma_cutter（挖掘工具）
- battery（电池）
- wire / carbon_fiber（电子材料）
- radiation_armor（防辐射护甲）
- 所有 workbench 配方

**唯一可用的合成**: building_block [handcraft] stone×3（无需电力）

### 2.2 降落仓不可拆卸

**严重程度**: 中 — 限制了基地迁移策略

测试了以下操作，全部失败：
- attack target_agent: "drop_pod" → 目标不存在
- mine 降落仓所在格 → 该格无可开采资源
- pickup → 该格无可拾取物品
- build workbench（在降落仓上）→ 工作台/熔炉/能源节点需建在当前格（但缺材料）

30 个配方中无任何拆解/重部署相关配方。

**结论**: 降落仓为固定建筑，无法移动、拆卸或重部署。电力和护盾范围 (Manhattan ≤3) 永久绑定在 (125,100)。

### 2.9 位置不同步 (Position Desync)

Tick 帧返回的位置经常滞后实际位置 1-20+ 格。例如：
- ember_status 显示 (125,142)，tick 帧显示 (120,143)
- 发送 8 次 move，tick 帧位置仅更新 2 格

**规则**: 永远不要用 tick 帧位置做 mine/chop/build 目标。始终用 ember_status 获取真实位置。

---

### 2.10 action_timeout

约 10-20% 的 ember_step 调用返回 action_timeout，即使 action 有效且服务器正常。action 未执行（不消耗能量，不改变位置），下个 tick 重试即可。

---

### 2.11 子 Agent (delegate_task) 建墙陷阱

多次出现子 agent 建墙把自己困住的情况：

**案例 1**: 子 agent 在 (120,142) 建 3 面墙，南/东面是铜矿脉（不可通行），完全被困。无法移动、挖掘或攻击墙壁。最终等辐射致死。

**案例 2**: 子 agent 建完全密封掩体无门，agent 在外面无法进入。

**修复建议**: 对子 agent 明确指令 "永远不要在第 4 面墙时站在掩体内部"。

## 六、建议

### 对游戏开发者

1. **修复 Iron Bootstrap Deadlock** — 方案之一：将 workbench 恢复为初始免费物品（dwad 版本的做法），或降低 workbench 合成需求（去掉 iron_ingot）
2. **石料矿点透明化** — 在视野中显示隐藏矿脉信息，避免玩家空挖浪费时间
3. **添加掩体门警告** — 建造完全密封建筑时提示 "警告：无门，无法进入"
4. **考虑添加降落仓重部署机制** — 允许玩家在游戏后期迁移基地

### 对 AI Agent

1. 建墙时严格遵循 "3 墙从内 + 第 4 墙从外 + 留门" 顺序
2. 始终用 ember_status 验证位置，不信任 tick 帧
3. action_timeout 直接重试，不改变策略
4. HP < 40 时立即撤退回护盾
5. 合成前确认电力（玩家在距离 ≤3 内）
